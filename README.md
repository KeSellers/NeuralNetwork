# NeuralNetwork

<b>Set up ur venv:</b>
  <p> python3 -m venv ./venv/</p>
<b> Acticate venv</b>
<p> cmd : venv\Scripts\activate.bat</p>

<p> UNIX or MacOs : source tutorial-env/bin/activate</p>

<b>Install dependencies</b>
<p>python -m pip install -r requirements.txt</p>
